Production Ready Private Cloud Software

Leveraging the power of OpenStack {{ site.openstack_version }}, Ansible and Linux Containers, Rackspace delivers a complete managed private cloud that is easy to deploy, manage and scale.

Open

The entirety of our design and tooling is open sourced. Everything from our Ansible Playbooks to our custom monitoring scripts that integrate with Rackspace Cloud Monitoring.

Experienced

We know OpenStack® and we have experience on how to deploy it. We've worked hard on distilling our best practices into an excellent deployment strategy for OpenStack®.

Scalable

Grow seamlessly with a cloud designed to scale to hundreds of nodes. We use Linux containers to independently scale each OpenStack service and enable seamless, in-place upgrades.

Included Services

Compute

Nova provisions and manages large networks of virtual machines.

Dashboard

Horizon provides administrators and users a graphical interface to access, provision, and automate cloud-based resources.

Object Storage

Swift provides redundant, scalable object storage using clusters of standardized servers.

Block Storage

Cinder provides persistent block level storage devices for use with OpenStack compute instances.

Networking

Neutron provides pluggable, scalable, API-driven network and IP management.

Image Service

Glance provides discovery, registration, and delivery services for disk and server images.

Identity Service

Keystone provides a central directory of users, mapped to the OpenStack services they can access.

Orchestration

Heat allows application developers to describe and automate the deployment of infrastructure via templates.
